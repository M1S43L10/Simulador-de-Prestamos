# 📝 Simulador de Préstamos - Entrega 2 Coderhouse

## 🚀 Descripción
Simulador interactivo de préstamos con sistema de amortización que permite:
- Cálculo de cuotas mensuales
- Períodos de gracia
- Pagos extraordinarios
- Comparación de escenarios
- Exportación de resultados

## 🛠 Tecnologías
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API

## 🔧 Instalación
1. Descargar el repositorio
2. Abrir `index.html` en cualquier navegador moderno
3. No requiere dependencias externas
